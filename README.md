# Python Perfect

## App Launching

#### Install Virtual Environment

```
$ pip3 install virtualenv
```

#### Activate Virtual Enviroment

```
$ python3 -m venv env
```

```
$ source env/bin/activate
```

#### Install and Run

```
$ pip3 install -r requirements.txt
```

```
##### On windows
$ set FLASK_APP=PythonPerfect.py
##### On mac/linux
$ export FLASK_APP=PythonPerfect.py
$ flask run
```
